    Build great HTML5 game dev tools at the HTML5 Tools Jam, January 8th + 9th, 2011 — Boston/worldwide »

    Paint a level in the left box, watch it show up in the game in the right box. Best run on a widescreen monitor! For extra awesome, try putting your browser in full screen mode. If your resolution is still too small, press ctrl and (-) to zoom your browser out. Try ctrl+Z/Y for undo/redo.

    Game controls: left/right arrow keys to move, Z to jump, C to respawn near the top-left of the screen.
